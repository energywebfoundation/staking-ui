Snapshot Process
1a.
Select a blocknumber range (start and end block numbers) from which a snapshot block is to be calculated
1b.
Obtain the blockhash of a block to use as a seed value. The EnergyWeb community commits to use the blockhash of a future block using SnapshotSeedRegistration contract SnapshotSeedRegistration contract
2.
Calculate a block number within the range using a pseudo-random number generator
3.
Generate a snapshot file with the block number as input.
